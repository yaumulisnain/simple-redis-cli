# SIMPLE REDIS CLI

## HOW TO USE
* Linux
    ```
    $ ./redis-linux-64

    ```
* Mac
    ```
    $ ./redis-mac-64

    ```
* Windows
    ```
    $ redis-windows-64.exe

    ```

## HELP
```
USAGE:
  apps [arguments...]
COMMANDS:
  SET [redis-key] [data-type] [value]
  GET [redis-key] [data-type]
DATA TYPES:
  STRING escape strings
  TIME RFC3339 format, ex: 2021-04-14T08:09:47Z
```
